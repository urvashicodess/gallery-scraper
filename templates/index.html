How to Use:

  1. You can either enter URLs manually in the text box or upload a CSV file containing gallery website URLs.
  2. If uploading a CSV, the file name will be displayed once selected.
  3. After entering URLs or uploading a file, click the Start Exploring button.
  4. Wait for the process to complete while the loading animation runs.
  5. Once the exploration is finished, a message will appear indicating that the download is ready.
  6. Click Download CSV to get the extracted data.
